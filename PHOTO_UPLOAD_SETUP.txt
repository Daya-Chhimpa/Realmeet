ANDROID PERMISSIONS SETUP:
--------------------------

1. Open: android/app/src/main/AndroidManifest.xml

2. Add these permissions BEFORE <application> tag:

<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />

3. Save the file

4. Rebuild the app:
   npm run android

===========================================

EMULATOR TESTING:
-----------------

For Android Emulator:
1. Open emulator
2. Click on camera icon (...)
3. Go to "Camera" settings
4. Enable "Virtual scene" or add test images
5. Or drag & drop images into emulator

For Gallery:
1. Open emulator
2. Drag & drop images into emulator
3. Images will be in Gallery/Photos app
4. Now you can select them in the app
